# TipToe

위치 기반 음악 추천 및 재생 앱



## 주요 기능

- 지도의 특정 포인트에 노래 저장소가 있습니다.
- 노래 저장소에는 해당 위치에서 사람들이 가장 많이 들은 순서대로 노래가 저장됩니다.
- 노래 저장소를 지나가면 가장 많이 들은 노래 1곡이 내 재생목록에 추가됩니다.
- 맘에 드는 노래가 있다면 재생 목록에 추가하고, 하트를 눌러 순위를 높일 수 있습니다.
